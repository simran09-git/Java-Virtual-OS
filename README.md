# Java-Virtual-OS
# 🖥️ Java Virtual OS

A feature-rich Virtual Operating System simulation built using Java Swing and AWT.  
This project creates a desktop-like environment with multiple built-in applications, personalization features, and an interactive graphical user interface.

## Features

###  Virtual Desktop Environment
- Full-screen desktop interface
- Custom wallpaper support
- Desktop application icons
- Taskbar with Start Menu
- Real-time digital clock

### Notes Application
- Create and save notes
- Delete saved notes
- Emoji support
- Notes stored using File Handling

### Calculator
- Addition
- Subtraction
- Multiplication
- Division

### File Manager
- Browse local files
- Open files directly from Virtual OS

### Music Player
- Select and play audio files
- Java sound API integration

###  Web Browser
- Quick browser access

###  Personalization Features
- Change desktop wallpaper
- Mood-based themes:
  -  Happy Mode
  -  Sleepy Mode
  -  Angry Mode
  -  Music Mode

###  Additional Tools
- Screenshot Capture
- System Information Viewer

##  Technologies Used

- Java
- Java Swing
- Java AWT
- Object-Oriented Programming (OOP)
- Event Handling
- File Handling
- Java Sound API
