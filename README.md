* Java Virtual OS

A desktop-based Virtual Operating System developed using Java that provides multiple utilities and a personalized mood-based experience through an interactive graphical user interface.

*Features

* File Manager
* Calculator
* Browser
* Notes Application
* Music Player
* Image Viewer

*Mood-Based Personalization

* Happy Mood 
* Angry Mood 
* Sleep Mood
* music mood
* Dynamic wallpaper changes based on the selected mood
* Mood-specific environment and user experience

* Technologies Used

* Java
* Swing/AWT
* Object-Oriented Programming (OOP)

* Project Overview

This Virtual OS combines multiple desktop utilities into a single application. In addition to productivity features such as a file manager, browser, calculator, and notes application, it includes a mood-based personalization system. Users can select different moods such as Happy, Angry, or Sleep, and the application automatically changes wallpapers and provides a customized experience based on the selected mood.

The project was developed to strengthen Java programming, GUI development, event handling, and Object-Oriented Programming concepts.
